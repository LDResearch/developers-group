# 天机阁开发者社区

开发者社区是连接天机阁与广大开源爱好者和区块链开发者的桥梁，是天机阁与社区一起组织和搭建的活跃技术交流平台。通过线下的全国巡回 Meetup、黑客马拉松、区块链峰会和线上的技术论坛、大咖直播、分析专栏等活动，我们在全球范围内打造开发者社区生态。同时，我们诚邀区块链、人工智能、物联网等优秀的开源社区和项目与我们一起共建务实、高效、优质、自发的开发者社区联盟。
